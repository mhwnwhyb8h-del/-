# 新疆旅行随身网页

这是一个手机网页版本，当前已内置 6月1日到6月11日的新疆行程、酒店、打包清单和旅行提醒。直接打开 `index.html` 就能用。

## 使用方式

1. 在电脑或手机浏览器打开 `index.html`。
2. 手机上可以用浏览器菜单选择“添加到主屏幕”。
3. 清单勾选状态会保存在当前浏览器里。

## 页面

- 行程：每日路线、亮点、车程和注意事项。
- 每日建议：每一天都包含推荐穿搭、最佳观赏时段和顺路时间安排。
- 酒店：按入住日期排列，可复制酒店名。
- 美食：每天午饭、晚饭分开推荐，每餐提供多个店铺备选。
- 清单：证件、衣物、电子设备、长途车程和应急物品，可勾选并保存在手机。
- 提醒：高原、天气、安检、独库路况和紧急电话。

## 修改数据

主要内容集中在 `script.js`：

- `tripDays`：修改每日行程。
  - `bestTime`：修改最佳观赏时段。
  - `outfit`：修改每日穿搭建议。
  - `schedule`：修改当天时间安排。
- `hotels`：修改酒店。
- `foodRecommendations`：修改每日午饭、晚饭店铺推荐。
- `checklist`：修改打包清单。
- `tips`：修改旅行提醒。

如果补充酒店详细地址、航班号、司机电话和每晚联系人，可以继续扩展成一键导航、拨打司机电话、今日行程提醒版。
